# peptide-encoding-dna-search
this script was written as part of my independent bioinformatics learning
This project identifies DNA substrings that encode a given amino acid sequence. It checks both original DNA strands and reverse complement strand, because peptide can be encoded on either strand of DNA

# purpose
purpose of this project is to practice python programming in bioinformatics context and to understand how DNA_sequences can encode amino acid sequences

# description
In biology each amino-acid is encoded by a codon, which is the group of three nucleotides. In this code the DNA strand is given  and our aim is to find the concrete region which is either in the original DNA or in complement strand and which is responsible for the encoding of this given protein
